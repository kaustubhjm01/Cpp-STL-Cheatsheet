C++ Standard Template Library CheatSheet
  • The Standard Template Library (STL) is a set of C++ template classes to provide common programming structures and functions such as lists, stacks, arrays, etc. It is a library of container classes, algorithms, and iterators.
    • 1. Vector in C++ STL
    Vectors are same as dynamic arrays with the ability to resize itself automatically when an element is inserted or deleted, with their storage being handled automatically by the container. Vector elements are placed in contiguous storage so that they can be accessed and traversed using iterators. In vectors,data is inserted at the end.
    • Declaring a vector of int type 
       vector<int> v;
    • begin() - Returns an iterator pointing to the first element in the vector 
       v.begin()
    • end() - Returns an iterator pointing to the theoretical element that follows the last element in the vector 
       v.end()
    • size() - Returns the number of elements in the vector. 
       v.size()
    • front() - Returns a reference to the first element in the vector 
       v.front()
    • back() - Returns a reference to the last element in the vector 
       v.back()
    • push_back() - It push the elements into a vector from the back 
       v.push_back()
    • pop_back() - It is used to pop or remove elements from a vector from the back 
       v.pop_back()
    • erase() - It is used to remove elements from a container from the specified position or range. 
       v.erase()
    • clear() - It is used to remove all the elements of the vector container 
       v.clear()
    2. Set in C++ STL
    Sets are a type of associative containers in which each element has to be unique, because the value of the element identifies it. The value of the element cannot be modified once it is added to the set, though it is possible to remove and add the modified value of that element.
    • Declaring a set of int type 
       set<int> s;
    • begin() - Returns an iterator to the first element in the set. 
       auto it = s.begin()
    • end() - Returns an iterator to the theoretical element that follows last element in the set. 
       auto it = s.end()
    • size() - Returns the number of elements in the set 
       s.size()
    • empty() - Returns whether the set is empty. 
       s.empty()
    • insert(ele) - Adds a new element 'i' to the set. 
       s.insert(ele)
    • find(ele) - Returns an iterator to the element 'ele' in the set if found, else returns the iterator to end. 
       auto it = s.find(ele)
    • count(ele) - Returns 1 or 0 based on the element ‘ele’ is present in the set or not 
       s.count(ele)
    3. Map in C++ STL
    Maps are associative containers that store elements in a mapped fashion. Each element has a key value and a mapped value. No two mapped values can have same key values.
    • Declaring a map of int type 
       map<int,int> mp;
    • begin() - Returns an iterator to the first element in the map. 
       auto it = mp.begin()
    • end() - Returns an iterator to the theoretical element that follows last element in the map. 
       auto it = mp.end()
    • size() - Returns the number of elements in the map 
       mp.size()
    • empty() - Returns whether the map is empty. 
       mp.empty()
    • clear() - Removes all the elements from the map. 
       mp.clear()
    • erase(ele) - Removes the key value ‘i’ from the map. 
       mp.erase(ele)
    • erase(iterator) - Removes the iterator from the map. 
       mp.erase(mp.begin())
    • insert({key,value}) - Insert elements with a particular key in the map container. . 
       mp.insert({i, i * i});
    4. Stack in C++ STL
    Stacks are a type of container adaptors with LIFO(Last In First Out) type of working, where a new element is added at one end and (top) an element is removed from that end only. Stack uses an encapsulated object of either vector or deque (by default) or list (sequential container class) as its underlying container, providing a specific set of member functions to access its elements.
    • Declaring a stack of int type 
       stack<int> s;
    • size() - Returns the size of the stack 
       s.size()
    • empty() - Returns whether the stack is empty. 
       s.empty()
    • top() - Returns a reference to the top most element of the stack 
       s.top()
    • push(ele) - Adds the element ‘ele’ at the top of the stack – 
       s.push(ele)
    • pop() - Deletes the top most element of the stack – 
       s.pop()
    5. Queue in C++ STL
    Queues are a type of container adaptors which operate in a first in first out (FIFO) type of arrangement. Elements are inserted at the back (end) and are deleted from the front. Queues use an encapsulated object of deque or list (sequential container class) as its underlying container, providing a specific set of member functions to access its elements.
    • Declaring a queue of int type 
       queue<int> q;
    • size() - Returns the size of the queue 
       q.size()
    • empty() - Returns whether the queue is empty. 
       q.empty()
    • push(ele) - Adds the element ‘ele’ at the end of the queue. 
       q.push(ele)
    • pop() - Deletes the first element of the queue. 
       q.pop()
    • front() - Returns a reference to the first element of the queue. 
       q.front()
    • back() - Returns a reference to the last element of the queue. 
       q.back()
    6. Priority Queue in C++ STL
    Priority queues are a type of container adapters, specifically designed such that the first element of the queue is the greatest of all elements in the queue and elements are in non increasing order (hence we can see that each element of the queue has a priority {fixed order})
    • Declaring a priority queue of int type 
       priority_queue<int> pq;
    • size() - Returns the size of the queue 
       pq.size()
    • empty() - Returns whether the queue is empty. 
       pq.empty()
    • push(ele) - Adds the element ‘ele’ at the end of the queue. 
       pq.push(ele)
    • pop() - Deletes the first element of the queue. 
       pq.pop()
    • top() - Returns a reference to the top most element of the queue. 
       pq.top()
    C++ STL Algorithms
    STL has an ocean of algorithms, Here I have mentioned Some of the most used algorithms on vectors and most useful one’s in Competitive Programming.
    • Declaring a vector of int type 
       vector<int> v;
    • sort(first_iterator,last_iterator) To sort the vector in the ascending order 
       sort(v.begin(),v.end())
    • reverse(first_iterator,last_iterator) Reversing the vector 
       reverse(v.begin(),v.end())
    • *max_element (first_iterator, last_iterator) To find the maximum element of a vector 
       *max_element (v.begin(), v.end())
    • *min_element (first_iterator, last_iterator) To find the minimum element of a vector 
       *min_element (v.begin(), v.end())
    • accumulate(first_iterator, last_iterator, initial value of sum) – To do the summation of vector elements 
       accumulate(v.begin(),v.end(),0)
      • ,
    • count(first_iterator, last_iterator,x) To count the occurrences of x in vector. 
       count(v.begin(), v.end(),5)
    • find(first_iterator, last_iterator, x) Returns an iterator to the first occurence of x in vector and points to last address of vector ((name_of_vector).end()) if element is not present in vector. 
       find(v.begin(), v.end(), 5)
